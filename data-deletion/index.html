사용자 데이터 삭제 안내

「슬기로운 경찰생활」 앱은 모든 데이터를 사용자의 기기에만 저장하며, 별도의 서버에 개인정보를 수집하거나 저장하지 않습니다.

데이터 삭제 방법

  • 앱 삭제: 기기에서 앱을 삭제하면 모든 데이터가 자동으로 삭제됩니다.
  • 앱 데이터 초기화: 설정 → 앱 → 슬기로운 경찰생활 → 저장공간 → 데이터 삭제

Facebook 로그인 데이터

Facebook 로그인을 통해 수집된 정보를 삭제하려면 위 방법으로 앱 데이터를 삭제한 후, Facebook 설정 → 앱 및 웹사이트에서 본 앱의 연결을 해제해 주세요.
